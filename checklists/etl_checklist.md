# ETL 检查清单

## 流程规范

| 检查项 | 检查内容 | 状态 |
|--------|----------|------|
| Extract | 是否正确抽取数据 | ✅/❌ |
| Transform | 是否正确转换数据 | ✅/❌ |
| Load | 是否正确加载数据 | ✅/❌ |
| Validate | 是否进行数据校验 | ✅/❌ |
| Log | 是否记录完整日志 | ✅/❌ |
| Finish | 是否正确清理资源 | ✅/❌ |

## 设计原则

| 检查项 | 检查内容 | 状态 |
|--------|----------|------|
| 幂等性 | 是否支持重复执行 | ✅/❌ |
| 可重跑 | 是否支持失败重跑 | ✅/❌ |
| 断点恢复 | 是否支持断点恢复 | 【待确认】 |
| 数据校验 | 是否执行数据校验 | ✅/❌ |
| 日志完整 | 是否记录完整日志 | ✅/❌ |

## 数据校验

| 检查项 | 检查内容 | 状态 |
|--------|----------|------|
| 数量校验 | 源数据与目标数据数量是否一致 | ✅/❌ |
| 主键校验 | 主键是否非空且唯一 | ✅/❌ |
| 字段校验 | 字段值是否符合规则 | ✅/❌ |
| 业务规则校验 | 业务逻辑是否正确 | ✅/❌ |

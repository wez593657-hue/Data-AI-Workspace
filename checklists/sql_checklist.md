# SQL 检查清单

## 基本规范

| 检查项 | 检查内容 | 状态 |
|--------|----------|------|
| SELECT * | 是否禁止使用 SELECT * | ✅/❌ |
| 显式字段 | 是否使用显式字段名 | ✅/❌ |
| 表别名 | 是否使用表别名 | ✅/❌ |
| JOIN 注释 | 是否说明关联关系 | ✅/❌ |
| WHERE 条件 | UPDATE/DELETE 是否带 WHERE | ✅/❌ |
| 临时表 | 是否合理使用临时表 | ✅/❌ |
| 字段注释 | 是否添加字段注释 | ✅/❌ |

## 命名规范

| 检查项 | 检查内容 | 状态 |
|--------|----------|------|
| 表名 | 是否符合命名规范 | ✅/❌ |
| 字段名 | 是否符合命名规范 | ✅/❌ |
| 别名 | 是否清晰 | ✅/❌ |

## 性能规范

| 检查项 | 检查内容 | 状态 |
|--------|----------|------|
| 索引使用 | WHERE 条件字段是否有索引 | ✅/❌ |
| JOIN 索引 | JOIN 关联字段是否有索引 | ✅/❌ |
| 函数包装 | 是否避免函数包装索引字段 | ✅/❌ |
| LIMIT | 是否限制返回数据量 | ✅/❌ |
| Explain Plan | 是否输出执行计划 | ✅/❌ |

## 安全规范

| 检查项 | 检查内容 | 状态 |
|--------|----------|------|
| SQL 注入 | 是否存在 SQL 注入风险 | ✅/❌ |
| 权限控制 | 是否合理控制权限 | ✅/❌ |

## 一致性检查

# 04 营销管理

## 营销管理


营销活动管理

营销活动管理-列表

功能描述

支持用户查看上级机构与本机构创建的营销活动列表并支持按条件筛选。

输入

支持通过“归属机构”、“活动名称”、“开始日期”、“任务状态”等条件进行查询，并提供“查询”、“重置”完成筛选。

处理

（一）操作说明

1. 用户进入营销活动管理列表页。

2. 用户可在查询条件区操作：

点击“查询”，系统按当前输入条件返回符合条件的客户列表结果。

点击“重置”，系统清空当前输入条件并恢复初始状态。

3. 用户可在列表左上角统一操作按钮区操作：

点击“新增”跳转新增页面，详见“营销活动管理-新增”章节。

点击“删除”展示删除弹窗，详见“营销活动管理-删除”章节。

选择一条或多条营销活动数据，点击“提交审核”按钮，发起审核流程，详见“营销活动管理-提交审核”章节。

4. 用户可在列表行内统一操作按钮区操作：

点击目标行数据的“编辑”按钮，右侧抽屉式弹出编辑侧边栏弹窗。

5. 用户点击活动名称，可进入活动视图查看执行情况和客户明细。

（二）规则与约束

1. 列表默认查询①本人创建的营销活动、②上级机构创建且审核通过的营销活动。

2. 默认按“创建时间”倒序（由近及远）排序。

3. 列表栏位包括复选框、活动名称、活动类型、开始日期、结束日期、归属机构、创建人、创建时间、任务状态、审批状态。

4. 任务状态枚举值：未开始、进行中、已结束。

①未开始状态显示规则：未到达活动开始时间的活动，任务状态显示“未开始”。

②进行中状态显示规则：到达开始日期，任务状态显示“进行中”。

③已结束状态显示规则：到达结束日期的活动任务状态显示“已结束”。

5. 同机构相同岗位不同人员支持查看彼此创建的营销活动，但不支持编辑、删除非本人创建的营销活动。仅支持编辑、删除本人创建的营销活动。

（三）权限说明

1、分支行业务岗和网点负责人可查看上级机构创建的营销活动；总分支行业务岗和网点负责人自行创建的活动仅对本机构及下级可见，对上级机构不可见。（根据客户的归属机构和归属客户经理确定哪些机构和客户经理能接受到营销活动）

2、营销活动创建权限：网点负责人、总/分行业务岗。

营销活动管理-新增（1.设置基础信息）

功能描述

支持用户按“1.设置基础信息”、“2.选择客群”、“3.分配营销人”、“4.选择指标”“5.完成”五步向导创建新增营销任务，提升营销活动创建、客户选择、人员分配和目标设置效率。

输入

处理

（一）操作说明

1、用户在营销活动列表页点击“新增”按钮，进入活动创建向导界面，页面顶部展示步骤节点提示向导：设置基础信息-选择客群-分配营销人-选择指标-完成。

2、默认展示“设置基础信息”页，支持用户设置活动的基本信息。

3、点击“下一步”校验表单必填项。未录入的必填项下方展示提示文案：必填项不允许为空。校验通过，进入下一页面“选择客群”。

（二）规则与约束

1、基础信息页面录入内容包括：活动名称、活动类型、开始日期、结束日期、统计截至日期、活动说明。

2、活动类型为“综合方案”、“价值提升”时，需要选择活动指标，活动类型为“客户维系”时，不需要选择活动指标。

3、营销活动关联的指标数据仅统计活动开始至统计截至日期的数据。统计截止时间默认联动反显结束时间，支持用户修改与调整。

（三）权限说明

1、支持总/分行业务岗、网点负责人新增营销活动，其他用户权限新增。

输入

处理

（一）操作说明

1、用户点击“新建客群”按钮，跳转至客群新增界面或用户选择目标客群。

2、若用户选择的目标客群存在无管户客户经理的客户，点击“下一步”，系统进入“分配营销人”页面。若目标客群不存在无管户客户经理的客户，点击“下一步”直接进入“选择指标”页面。

3、选择客群时，仅支持单选，不支持多选。

4、营销活动关联选择客群时，需要单独建立客群快照，与客群管理模块彻底解耦。不影响客群正常的编辑与删除操作。

营销活动管理-新增（3.分配营销人）

功能描述

支持用户对营销活动关联的客群中无管户客户经理的客户进行营销人分配。

输入

客户列表的筛选条件

客户列表左上角统一按钮操作区

客户经理列表的筛选项

页面底部按钮

处理

（一）操作说明

1、用户在“选择客群”页面，点击“下一步”按钮，系统从右侧抽屉是弹出“分配营销人”页面。

2、在“分配营销人”页面默认反显客群中无管户经理的客户列表，弹窗顶部筛选区支持通过客户名称筛选客户。弹窗列表左上角统一操作区展示“选择营销人”按钮。选中待分配客户后点击“选择营销人”按钮，从右侧抽屉式弹出辖下客户经理列表。

3、在辖下客户经理列表页面，支持根据客户经理姓名及工号筛选。选择客户经理点确认按钮，需进行二次确认弹窗，提示文案：“确认将{X}位客户分配给XXX客户经理吗？”。点击确认返回客户列表并刷新，可继续进行分配。点击二次确认弹窗的“取消”按钮，返回客户列表，可继续进行分配。

4、用户完成营销人分配后，点击“下一步”按钮。进入“选择指标”页面。

（二）规则与约束

1、支持用户单个或批量的将客户分配给营销人，每次分配操作支持分配给单个客户经理。

2、分配营销人这一步不做强制操作，用户可以分配营销人，也可以不分配营销人。

（三）权限说明

同营销活动管理-新增（1设置基础信息）章节的权限说明。

营销活动视图-任务监控分析

功能描述

支持用户按任务维度查看营销任务监控分析结果。

输入

略

处理

（一）操作说明

1、用户进入营销活动详情后，系统默认展示“任务监控分析”tab，任务监控分析界面由上至下分为两模块展示：任务完成情况分析、任务关联指标趋势分析。

2、任务关联指标趋势分析默认展示首个指标，若存在多个指标，可点击其他指标的名称进行切换展示不同指标的趋势分析。

3、支持通过切片器在“日”“月”“季”之间切换趋势分析粒度。

4、用户点击“导出”后，可导出：《任务完成情况分析.xls》。

（二）规则与约束

1、任务完成情况分析以列表形式展示。列表栏位包括：机构名称、客户数、基准值、当前值、增量、增长率、接触覆盖率、拟支持客户数、进一步营销客户数、不予支持客户数、营销成功率。若活动关联多个指标，则每个指标都包括基准值、当前值、增量、增长率，支持任务完成情况分析表格横向扩展。

2、任务关联指标趋势分析以折线图展示，默认展示首个指标的趋势分析。当存在多个指标时，默认展示第一个指标（按指标编号排序），允许用户切换指标趋势分析切片器枚举值为“日”“月”“季”，默认值为“日”。

3、默认按照机构号升序排序。

（三）权限说明

1、支持总/分行管理用户、网点负责人查看本人创建或上级机构下发的营销活动详情

2、有权限（可配置不同角色的导出权限）的用户支持导出。

输出

系统展示机构维度分析结果、趋势分析结果及导出处理结果。

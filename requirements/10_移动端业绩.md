# 10 移动端业绩

## 业绩


业绩战报

功能描述

支持用户查看不同统计期间内的重点产品业绩概览，帮助用户快速掌握本人、机构的阶段性经营成果并支撑经营分析。

输入

用户进入“业绩”页面后，系统加载默认“月度”下的业绩概览结果。

用户可切换“统计期间”查看月度、季度、年度，不同口径下的业绩战报数据。

处理

（一）操作说明

1. 用户进入“业绩”页面后，系统默认加载当前统计期间（月度）的业绩概览结果。

2. 用户切换月度、季度、年度后，系统刷新对应统计期间的战报数据。

（二）规则与约束

1. 统计期间支持“月度”、“季度”、“年度”三类口径。

2. 统计范围包括“存款”、“理财”、“保险”、“贷款”四类业绩销量。

3. 机构、客户经理的产品销售数据按管户口径进行汇总统计。

4. 管理用户进入页面时，展示对应的统计期间，“存款”、“理财”、“保险”、“贷款”四类业绩销量，支持下钻，按照总行下钻查看分行、分行下钻查看网点、网点下钻查看客户经理、客户经理下钻查看客户的链路。

5. 客户经理用户进入页面时，展示对应的统计期间，“存款”、“理财”、“保险”、“贷款”四类业绩销量，支持下钻，查看客户的销量。

6. 机构层级、客户经理层级的产品销量，单位为万元，单一客户的产品销量单位为元。

（三）权限说明

1. 客户经理可查看本人名下管户客户的业绩战报数据。

2. 管理用户可查看其权限范围内机构业绩战报数据。

输出

系统输出当前统计期间下的业绩战报概览信息。

系统输出机构层级统计的列表信息。

系统输出客户经理层级统计的列表信息。

英雄榜

功能描述

支持总/分/支行不同层级用户查看机构和客户经理在多项经营指标下的排名，帮助用户快速识别领先和落后对象并支撑经营督导与绩效管理。

输入

用户进入“业绩”页面后，系统加载默认统计期间（月度）分行榜单、网点榜单及客户经理榜单。

用户可切换统计期间（年度、季度、月度）查看分行榜单、网点榜单及客户经理榜单。

处理

（一）操作说明

1. 用户进入“英雄榜”页面后，系统默认加载当前统计期间（默认：月度）。

2. 用户可切换 “月度” 、“季度”、 “年度”查看对应统计期间的榜单结果，在移动端支持“分行榜单”、“网点榜单”、“客户经理榜单”榜单的切换。

3. 根据用户权限，查看分行榜单、网点榜单及客户经理榜单排名。

4. 用户可切换不同指标，系统刷新对应榜单下的排名。

5. 系统同时展示前列排名和末位排名，便于用户识别领先对象和短板对象。

（二）规则与约束

1. 统计期间支持“月度” 、“季度”、 “年度”三类统计口径。

2. 统计榜单支持“分行榜单”、“网点榜单”、“客户经理榜单”三类对象。

3. 指标包括“AUM余额新增”、“定期存款余额新增”、“理财余额新增”、“活动量（总计、面访、电访、企微、短信）”、“潜力提升户数”，支持“分行榜单”、“网点榜单”、“客户经理榜单”三个榜单独立切换指标，独立排行。指标若选择活动量，则展示总计、面访、电访、企微、短信多个分项统计指标。

4. 指标口径说明

AUM余额新增：根据统计期间，统计较上期（如月度，为较上月）的AUM增量。

定期存款余额新增：根据统计期间，统计较上期（如月度，为较上月）的定期存款增量。

理财余额新增：根据统计期间，统计较上期（如月度，为较上月）的理财增量。

活动量：根据统计期间，统计当期（如月度，为当月）的活动量增量。

潜力提升户数：根据统计期间，统计当期（如月度，为当月）的潜力提升户数增量。统计期间的潜力提升户数的口径按重点经营视图-潜力提升章节中的月均达标口径。

5. 用户为总行管理用户，默认展示“分行榜单”，客切换“网点榜单”、“客户经理榜单”三个榜单，展示规则为排名默认展示前 TOP10 和后 TOP10；若样本数小于 20，则全部展示。

6. 用户为分行管理用户，默认展示“分行榜单”、客切换“网点榜单”、“客户经理榜单”三个榜单，展示规则为排名默认展示前 TOP10 和后 TOP10；若样本数小于 20，则全部展示。若本分行不在前后20名，则增加一条本分行排行数据。

7. 用户为网点管理用户，默认展示“网点榜单”、“客户经理榜单”两个个榜单并且可切换，展示规则为排名默认展示前 TOP10 和后 TOP10；若样本数小于 20，则全部展示。若本网点不在前后20名，则增加一条本网点排行数据。

8. 用户为客户经理，界面展示 “客户经理榜单”，展示规则为排名默认展示前 TOP10 和后 TOP10；若样本数小于 20，则全部展示。若本人不在前后20名，则增加一条本人排行数据。

9. 排名结果需按所选指标值排序。

（三）权限说明

1. 总行用户可查看分行榜单、网点榜单、客户经理榜单。

2. 分行用户可查看分行自身榜单，并查看辖下网点榜单、客户经理榜单。

3. 网点用户可查看本网点在全行网点中的榜单，并查看辖下客户经理榜单。

4. 客户经理仅可查看本人在全行客户经理中的榜单

输出

英雄榜-分行排行

英雄榜-网点排行

英雄榜-客户经理排行
